The dataset, derived from the SQLite database, contains variables:

'data.frame':  18 variables:
* id        : int
* tid       : chr
* username  : chr
* statuses  : chr
* since     : chr
* followers : int
* friends   : int
* location  : chr
* utc_offset: int
* created_at: chr
* content   : chr
* geo       : chr
* meta      : chr
* hashtags  : chr
* urls      : chr
* media     : chr
* source    : chr
* lang      : chr
